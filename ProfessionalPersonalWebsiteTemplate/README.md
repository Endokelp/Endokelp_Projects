# Portfolio site template

Vite + React + TypeScript + Tailwind. I made this so I don’t start from zero every time I want a personal site.

## Run it

```bash
npm install
npm run dev
```

Then open whatever URL Vite prints (usually `http://localhost:5173`).

## What to change

- Swap the copy in `src/components/*`—a lot of it is still dummy text.
- `index.html` title/meta.
- Put a real photo in `public` if you want one (right now the About section is a gray box).
- Contact links: email + GitHub at minimum; fix or delete anything you don’t use.

## Build

```bash
npm run build
```

Output lands in `dist/`. Upload that folder to whatever host you like.

No warranty. If something breaks after a dependency bump, that’s normal.
